# blog

## 技术栈
*  前端 Vue + vue-router + element-ui，使用TypeScript语言编写
*  后端 nodejs + mysql

因为项目比较小，没有使用vuex，需要的全局数据用类封装了（这种方案亦能达到状态管理的效果，原理是使用类的get实现，get的值改变也能触发vue视图相应的更新）

## 简介
个人毕业设计做的项目，参考了很多商业博客网站，基本功能都已经实现，markdown编辑器、后台管理、搜索、评论。
